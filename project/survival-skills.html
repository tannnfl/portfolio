The Survival Skills Game

Solo Developer · Physics Game · 2026
Solo Dev Physics Play on itch.io ↗

Overview

"Hey kid, I know it's hard when you first start taking care of yourself. We all went through it once, so know that you're not alone. Here is a tutorial I made for you to practise the very important survival skills to live alone. Challenge yourself until you're S-graded proficient in performing these skills. Remember, I am proud of you."

  • 3 mini-games framed as a "survival tutorial" for living alone — the title is deliberate irony
  • Physics-driven chaos is the point; the game never takes itself seriously for a single second
  • Solo-developed and shipped as a browser-playable build on itch.io

My Role

  • Independently designed all 3 mini-games from concept to ship
  • Implemented all gameplay, physics, and UI
  • Created all visual assets
  • Shipped as a browser-playable build on itch.io

Design Notes

  • Scope discipline: ship something complete and playable rather than reach for scale
  • Comedic framing gave the mini-games permission to be messy — and that messiness became the point
  • More levels in development based on player requests
