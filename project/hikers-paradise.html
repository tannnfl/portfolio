Hiker's Paradise

Solo Developer · Procedural Terrain · 2025
Game Prototype Procedural Generation Unity Solo Dev

Overview

A national-park management simulator in which players sculpt infinite procedurally generated mountains and maintain terrain in real time. Six tool modes — dig, raise, flatten, smooth, paint, and tree edit — give players direct authorship over the landscape.

My Role

  • Designed and implemented all six terrain-editing tools
  • Built the procedural mesh system: vertex and triangle generation, Perlin-noise height shaping with edge falloff
  • Authored an intersection shader for real-time brush feedback
  • Implemented slope- and height-based vertex coloring with procedural tree placement
  • Designed and built the full UI flow

Design Notes

The challenge was keeping terrain editing tactile and immediate — each brush stroke had to feel responsive without causing frame drops. Future development targets trail placement and a lightweight economy layer.
