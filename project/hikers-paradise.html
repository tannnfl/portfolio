Hiker's Paradise: Demo

Solo Developer · Procedural Terrain Generation
Game Prototype Procedural Generation Unity

Overview

A simulation game about managing a national park. The demo lets players generate infinite mountains and edit terrain with tools: dig, raise, flatten, smoothen, paint colors, and edit trees.

Technical Process

  • Generate mesh with vertices and triangles
  • Shape via vertex height modification using multiple Perlin noise layers and edge falloff
  • Intersection shader for the brush tool
  • Slope and height-based vertex color mapping; Perlin noise tree generation
  • UI and terrain editing system
  • Future plans: trail system and economy system
