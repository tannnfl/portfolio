POPLife Data Browser

Tool Developer · Unity Editor Extension · C# · 2025–Present
Tool Dev Unity Editor C# ← View Game ← View GDD

Overview

POPLife Data Browser is a custom Unity Editor window built to author and manage all ScriptableObject-based game data for POPLife. With 86 shelves, 66 quests, 40+ NPCs, and multiple interlocking event systems spread across hundreds of assets in a deep project hierarchy, the default Unity Inspector was too slow to work with at scale.

The browser aggregates every asset type into a single searchable panel — with live editing, inline cross-references between SOs, and a "Select in Project" shortcut when you do need to navigate to the file. It also introduces relationships that the default Inspector can't express: Quest Chains link Activation and Completion Markers across multiple quests, enabling nested trigger logic for branching quest progressions that couldn't be authored any other way.

Impact

  • Eliminated ~80% of time spent on narrative implementation — dialogue actions, op guides, and quest chains can be authored entirely without leaving the browser
  • Eliminated ~80% of iteration time on balance and quest design — shelf formula stats, quest conditions, and reward chains are all visible in one view
  • Removed folder-navigation overhead for a 7+ person team working across hundreds of ScriptableObjects
  • Quest Chains tab introduced a new data relationship not representable in plain Inspector fields — activation markers cascade across SO boundaries to support branching, multi-step quest progressions

Tabs

  • Shelves — 86 assets. Displays formula-computed stats (Stock Fee, Profit, Price, Maint. Fee, Appeal) across all 5 upgrade levels at a glance. Footprint pattern editor included.
  • Quests — 66 assets. Full quest authoring: type, giver, activation/completion markers, reward sets, conditions, and display text in one scrollable form.
  • Codex — 86 assets. In-game item encyclopedia — description and usage instruction authoring, mirroring the in-game Codex UI.
  • Buffers — 8 assets. Event buffers with trigger type, occurrence probability, duration, economy modifier payload, and optional activation dialogue.
  • Season Events — 11 assets. Calendar-based seasonal events positioned by season and day-in-season.
  • News Libraries — 22 items. In-game news feed content authoring with move-up/move-down ordering and character count.
  • Op Guides — 12 assets. Multi-page tutorial guides with marker-based activation and per-page screenshot + body text.
  • Dialogue Actions — 26 assets. Marker-triggered conversation sequencing with execution order and Lua scripting support.
  • Quest Chains — 68 quests across chains. Hierarchical tree view of quest progressions; each node exposes Trigger Chain with Activation/Completion Markers, enabling nested SO-to-SO trigger logic not representable in the default Inspector.
