Oh No! Neko Ate My Eyes!

Solo Developer · Audio Design · 2024
Game Prototype Audio Design Unity FMOD Solo Dev Play on itch.io ↗

Overview

A spatial audio game in which the player has lost their sight and must locate five cats hidden in darkness using only 3D sound. Players whistle to elicit responses from nearby cats, navigate via audio proximity cues, and interact with objects — cat, radio, or window — to identify them.

My Role

  • Designed the full audio-navigation system in Unity with FMOD
  • Implemented 3D spatial sound for all interactive elements
  • Built the beeping proximity indicator system and text feedback overlay
  • Designed the level blockout and interaction framework
