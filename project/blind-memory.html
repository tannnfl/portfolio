Blind Memory

Programmer · Technical Artist · Audio Engineer · 2024
Game Prototype Audio Design Unity FMOD Play on itch.io ↗

Overview

A 2D game in which players move through an environment, frame photographs, and watch them auto-post to an in-game blog — part photography puzzle, part atmospheric narrative. Built in Unity with FMOD audio integration.

My Role

  • Led programming, technical art, audio engineering, and project management
  • Designed and implemented the photography and automatic blog-posting mechanics
  • Created background art for the main scene and newspaper graphics for the ending sequence
  • Collaborated with an artist and a narrative designer on a three-person team

Design Notes

The photography-blog loop creates a passive record of the player's journey — every image is both action and artifact. The demo establishes the core mechanic; future iterations will expand into a 3D environment with richer world-building and a more refined visual feedback loop.
