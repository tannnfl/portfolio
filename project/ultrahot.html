Ultrahot Demo

Solo Developer (Design + Programming)
Game Prototype Unity 3D Solo Dev

Overview

A top-down action game inspired by Superhot. Players escape enemies and eliminate them. Core mechanic: time scales with player movement — faster when moving, nearly stopped when standing still.

Technical Details

  • Unity 3D with Navmesh for enemy navigation
  • Time manipulation via Time.timeScale
